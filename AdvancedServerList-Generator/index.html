Common Placeholders

${player name} - Player name

${player isBanned} - Is player banned

${server playersOnline} - Online players

${server playersMax} - Max players

${server tps} - Server TPS

Tips & Tricks

  • Higher priority profiles are evaluated first
  • Use conditions to show different MOTDs for different scenarios
  • Favicon URLs/players are cached for performance
  • Custom player count text shows server as "outdated"
  • Use 'random' as favicon to randomly select from your favicons folder
